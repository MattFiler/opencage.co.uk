Loadscreen Movies

To edit ammo configurations in OpenCAGE navigate to Edit > Configurations > Core Game > Loadscreen Movies.

From here you will be presented with a variety of options for movie playlists used in the loadscreen, relating to the selected playlist in the dropdown at the top of the window.

Select a playlist, modify its properties, and press Save to update them in-game.

Read on to see a list of available properties with information about each.

Playlist Settings

  • Allow Skip bool
    • If checked the movie playlist be skipped.
  • End Playlist When Loading Finished bool
    • If checked the playlist will be exited after the level load has completed.
  • Shuffle bool
    • If checked the playlist will run in any random order.
  • Loop bool
    • If checked the playlist will loop until an exit condition is met (skipped if 'Allow Skip', or level loaded if 'End Playlist When Loading Finished').

Movies In Playlist

Here the list of movies to play within the playlist are defined.

To add a new movie, select 'Add New'. You should locate a movie file within the game's DATA/UI/MOVIES directory.

To remove a movie select 'Remove'. You can also re-order the playlist using 'Move Up' and 'Move Down', but note that if 'Shuffle' is checked (see above) the order will be randomised.
