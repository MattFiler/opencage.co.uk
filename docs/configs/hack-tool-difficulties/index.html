Hack Tool Difficulties

To edit the hack tool difficulties in OpenCAGE navigate to Edit > Configurations > Core Game > Hack Tool Difficulties.

From here you will be presented with the ability to modify minigame configurations for each difficulty level and the maximum difficulty configuration used per hack tool level.

Read on to see a list of available properties with information about each.

Max Difficulties

  • Tool Level 0 combobox
    • Defines the maximum difficulty level supported by hack tool level 0 (the tool level can be set by SetGatingToolLevel).
  • Tool Level 1 combobox
    • Same as above, but for level 1.
  • Tool Level 2 combobox
    • Same as above, but for level 2.
  • Tool Level 3 combobox
    • Same as above, but for level 3.
  • Tool Level 99 combobox
    • Same as above, but for level 99.

Difficulty Settings

This defines the configuration for each difficulty level, select one from the dropdown to see the associated values.

  • Inner Selection Angle (Degrees) float Minimum: 0 Maximum: 360
    • Defines the inner selection angle for the hacking minigame's tuning in degrees.
  • Outer Selection Angle (Degrees) float Minimum: 0 Maximum: 360
    • Defines the outer selection angle for the hacking minigame's tuning in degrees.
  • Selection Angle Increase (Degrees) float Minimum: 0 Maximum: 360
    • Defines the increase to apply to the tuning selection angle after the grace time (defined in HACKINGV4.GFX).
  • Number Of Rounds int Minimum: 0
    • Defines the number of rounds the player must complete in the hacking game.
  • Length Of Keycode int Minimum: 0 Maximum: 8
    • The length of the keycode for the symbol matching game. Note the actual total is this value added to 'Number Of Alarms'.
  • Length Of Keycode int Minimum: 0 Maximum: 8
    • Defines the number of 'alarmed' symbols in the matching game. This functionality is deprecated but still used to define the full number of symbols in the matching game (see above).
  • Timer Countdown In Seconds int Minimum: 0 Maximum: 8
    • Defines the amount of time the player has to complete the minigame in seconds.
